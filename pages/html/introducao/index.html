Breve resumo e introdução ao HTML

Emilayne Rodrigues

A linguagem HTML é a linguagem para a escrita de páginas Web. HTML é o acrônimo de HyperText Markup Language (Linguagem de Marcação de HiperTexto). Ser uma linguagem de hipertexto significa ser capaz de escrever páginas interligadas (contendo ligações entre as páginas; quando uma ligação é ativada, outra página deve ser apresentada). Ser uma linguagem de marcação significa que a página é construída a partir de símbolos utilizados para marcar como o conteúdo da página deverá ser apresentado.

Os navegadores de HTML são programas capazes de apresentar uma página HTML. Os principais navegadores são o Internet Explorer e o Netscape. Os editores de HTML são programas capazes de escrever uma página em HTML. Microsoft FrontPage e o Netscape Composer são exemplos de bons editores de HTML. Como o código de uma página HTML é apenas um texto simples, qualquer programa capaz de escrever textos pode ser utilizado para editar uma página HTML - neste curso, o Editor HTML e o Bloco de Notas (do Windows) serão suficientes para você estudar esta linguagem e construir suas próprias páginas.

A linguagem HTML possui basicamente quatro conceitos fundamentais: elemento, etiqueta, atributo e valor. Um elemento é uma estrutura (como parágrafo, lista, tabela, imagem etc.) utilizada na composição de uma página HTML. As etiquetas delimitam o início e o fim de um determinado elemento. Um atributo é uma característica do elemento que pode ser configurado através da especificação de um determinado valor.

A estrutura de um documento HTML é composta por três partes: a estrutura principal (delimitada pelas etiquetas < html> e < /html> ), o cabeçalho (delimitado pelas etiquetas < head> e < /head> ) e o corpo (delimitado pelas etiquetas < body> e < /body> ). Todo o conteúdo de uma página HTML é escrito dentro do corpo do documento - ou seja, praticamente todos os elementos abordados nos próximos capítulos deste curso são escritos dentro de < body> e < /body> .

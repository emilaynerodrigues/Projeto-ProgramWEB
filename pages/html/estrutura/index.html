Estrutura padrão do HTML

Emilayne Rodrigues

Assim como qualquer outro tipo de documento, o HTML tem sua estrutura padrão e sua forma própria forma de ser escrito e iniciado. Está estrutura em si é imutável, não importa a complexidade do documento ou o tamanho, ela sempre estará lá.

Cada editor de texto tem atalhos que facilitam a rápida inserção desta estrutura, por exemplo no Visual Studio Code você conta com a ajuda do Emmet, que tem duas formas rápidas de construir esta estrutura:

  1. Digitando html:5 + enter; ou
  2. Inserido o ponto de exclamação (!) + enter

Com toda essa facilidade não é preciso decorar ou escrever passo por passo todas as vezes que formos iniciar um documento HTML, porém é fundamental que saibamos como escrevê-la manualmente e a diferença que faz cada elemento dessa estrutura.
